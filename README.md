# Trading bot

## Что тут есть
- базовая стратегия сигнала
- поддержка 10 популярных монет
- файл запуска бота
- файл конфигурации

## Как запустить
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите бота:
   ```bash
   python run_bot.py
   ```

## Где что менять
- В файле config.py поставьте свои API ключи и токен Telegram.
- В файле run_bot.py замените sample_prices на реальные данные с биржи.
- В файле trading_bot/strategy.py можно изменить правила стратегии.

## Важное замечание
Это учебный пример. Для реальной торговли лучше тестировать сначала на демо-счёте.
